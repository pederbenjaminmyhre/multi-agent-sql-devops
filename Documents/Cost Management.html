Cost Measures Implemented

1. Scale-to-Zero Compute

Azure Container Apps is configured with minReplicas: 0 and maxReplicas: 2. When no requests are in flight, the container is deallocated entirely. There is zero compute cost during idle periods. Cold starts add 2-5 seconds, which is acceptable since the LLM calls themselves take 10-30 seconds.

2. Serverless Database

Cosmos DB uses the Serverless capacity mode instead of Provisioned Throughput. Provisioned mode requires paying for reserved RU/s even when idle. Serverless charges only for the request units actually consumed, with no minimum. At portfolio traffic this reduces database costs from ~$24/month to under $1/month.

3. Basic-Tier Container Registry

ACR Basic ($5/month) provides 10 GB storage and sufficient throughput for a single-image portfolio project. The Standard tier ($20/month) and Premium tier ($50/month) offer geo-replication and other features that are unnecessary here.

4. Iteration Cap on Critique Loop

Each critique-revise loop iteration costs 4 LLM calls. Without a cap, a pathologically adversarial input could trigger unlimited loops. The configurable max_iterations (default: 3) limits each review to a maximum of 12 LLM calls, bounding the per-review OpenAI cost at roughly $0.50.

5. Schema Context Injection (Not Tool Calls)

Schema information is injected directly into prompts rather than retrieved via tool calls. Each tool call that the LLM initiates adds a round-trip (prompt → tool call → tool response → LLM continuation), doubling the token usage. Direct injection eliminates this overhead.

Best Practices for Ongoing Cost Control

Set an Azure Budget Alert

Create a budget in the Azure Portal to receive email alerts when spending approaches a threshold:

  1. Navigate to Cost Management + Billing in the Azure Portal.
  2. Select Budgets → Add.
  3. Set a monthly budget (e.g., $15) scoped to rg-sql-devops-portfolio.
  4. Configure alerts at 80% and 100% of the budget.

Tear Down When Not Demoing

If the project will be idle for an extended period, delete the resource group to eliminate all costs:

az group delete --name rg-sql-devops-portfolio --yes --no-wait

Redeploying takes about 5 minutes with bash infra/deploy.sh. The code, configuration, and Docker image can be rebuilt from the repository at any time.

Monitor Actual Spend

Check real costs in the Azure Portal:

  1. Navigate to Cost Management → Cost analysis.
  2. Scope to rg-sql-devops-portfolio.
  3. Group by Resource to see per-service breakdown.

Monitor OpenAI Usage

Track LLM costs separately at platform.openai.com/usage. Set a monthly spend limit in the OpenAI dashboard under Settings → Limits.
